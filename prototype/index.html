Correctness by Construction: Das Modell ist nach jeder Operation strukturell gültig (Stufe A). Über die „+"-Punkte werden ausschließlich korrektheitserhaltende Operationen angeboten – z. B. fügt eine Verzweigung immer einen vollständigen, balancierten Block (Split + Join) ein. Freigeben (Stufe B) ist erst möglich, wenn alle Bindungen (Dienst, Bearbeiter, Daten) vollständig sind.
